Introduction to Parallel Programming

Parallel computing is a type of computation which multiple processing elements collabrate to process a task simultaneously. Higer performance is aimed for using parallel computing techniques comparing with sequencial processing. By breaking up the problem into multiple parts, each processing element can execute the part with others simultaneously.

What is Blockly?

Blockly is an open-sourced, client-sided JavaScript library that adds a visual code editor to web and mobile apps.

So Blockly is a visual programming tool made up of blocks. You can use the drag and drop features to make your own programme. The blocks structure you make can be used to generate real JavaScript, Python, PHP, Lua and Dart code. Extensions for other language can be found on Github. Here is the Java Extension of Blockly.
